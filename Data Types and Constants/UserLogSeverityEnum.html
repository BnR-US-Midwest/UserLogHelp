UserLogSeverityEnum

USERLOG_SEVERITY_DEBUG and USERLOG_SEVERITY_CRITICAL are extensions of ArEventLog's severities.

Parameter Value Description
USERLOG_SEVERITY_DEBUG -1 ArEventLog Information - Developer information
USERLOG_SEVERITY_SUCCESS 0 ArEventLog Success - Completion
USERLOG_SEVERITY_INFORMATION 1 ArEventLog Information - User information
USERLOG_SEVERITY_WARNING 2 ArEventLog Warning - The application may continue
USERLOG_SEVERITY_ERROR 3 ArEventLog Error - The application is recoverable
USERLOG_SEVERITY_CRITICAL 4 ArEventLog Error - The application is unrecoverable
