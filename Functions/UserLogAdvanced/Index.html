UserLogAdvanced

Write to the User logbook with runtime data.

If successful, this function returns the record ID (type ArEventLogRecordIDType) of the event created in the User logbook. If an error occurred, an error message is written to the User logbook and the function returns zero.

Function

Interface

I/O Parameter Data Type Description
IN Severity DINT Use UserLogSeverityEnum or arEVENTLOG_SEVERITY constants
IN Code UINT 0..65535
IN Message STRING[0] Event description with optional format specifiers
IN Values UserLogFormatType Format values to replace specifiers (%b, %f, %i, %s)
OUT Return UDINT Record ID if successful, zero otherwise
