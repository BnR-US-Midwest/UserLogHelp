UserLogCreate

Create a custom logbook in an Init subroutine.

This function returns the status ID from the internal ArEventLogCreate function block call.

Function

Interface

I/O Parameter Data Type Description
IN Name STRING[0] Name of logbook limited to 10 characters
IN Size UDINT Size of logbook in bytes (minimum 4096)
OUT Return DINT Returns ArEventLogCreate status ID
