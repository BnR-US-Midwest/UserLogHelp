UserLogEventText

Write to any user logbook through binary-encoded data and event texts.

If successful, this function returns the record ID (type ArEventLogRecordIDType) of the event created. If an error occurred, an error message is written to the User logbook and the function returns zero.

Function

Interface

I/O Parameter Data Type Description
IN Logbook STRING[0] Name of logbook
IN Event DINT Event ID (match Text ID)
IN Origin ArEventLogRecordIDType (Optional) Origin record ID
IN Object STRING[0] (Optional) Object name
IN Message STRING[0] Event description with optional format specifiers
IN Values UserLogFormatType Format values to replace specifiers (%b, %f, %i, %s)
OUT Return UDINT Record ID if successful, zero otherwise
