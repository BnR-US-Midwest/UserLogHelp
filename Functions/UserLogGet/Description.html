Description

These functions allow the user to decode an 32-bit event ID into severity, facility, and code. See ArEventLogMakeEventID function for creating an ID.

Example

IEC Structured Text

    VAR
        Severity : USINT;
        Facility : UINT;
        Code : UINT;
    END_VAR
  
    Severity := UserLogGetSeverity(USERLOG_EXAMPLE_TEXT_ID);
    Facility := UserLogGetFacility(USERLOG_EXAMPLE_TEXT_ID);
    Code := UserLogGetCode(USERLOG_EXAMPLE_TEXT_ID);

"Severity" is 1 (arEVENTLOG_SEVERITY_INFO). "Facility" is 330 (USERLOG_FACILITY). "Code" is 20000.

ANSI C

    VAR
        Severity : USINT;
        Facility : UINT;
        Code : UINT;
    END_VAR
  
    Severity = UserLogGetSeverity(USERLOG_EXAMPLE_TEXT_ID);
    Facility = UserLogGetFacility(USERLOG_EXAMPLE_TEXT_ID);
    Code = UserLogGetCode(USERLOG_EXAMPLE_TEXT_ID);

"Severity" is 1 (arEVENTLOG_SEVERITY_INFO). "Facility" is 330 (USERLOG_FACILITY). "Code" is 20000.
