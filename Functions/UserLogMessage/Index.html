UserLogMessage

Write to logbook synchronously. Returns record ID if successful, zero otherwise.

Function

Interface

I/O Parameter Data Type Description
IN logbook STRING[0] Name of logbook
IN severity DINT Use UserLogSeverityEnum or arEVENTLOG_SEVERITY constants
IN facility UINT 0..15 application area 16..4095 3rd-party device area
IN code UINT 0..65535 unique code
IN origin ArEventLogRecordIDType (Optional) Origin record ID
IN object STRING[0] (Optional) Object name
IN message STRING[0] ASCII data message
IN args UserLogFormatType Format arguments
OUT return UDINT Record ID if successful, zero otherwise
