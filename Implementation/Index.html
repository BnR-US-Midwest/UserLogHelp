Implementation

The UserLog library's source code is written in C. The implementation utilizes the ArEventLog library's function blocks to create, reference, and write to user logbooks. Instead of function blocks which require multiple calls, UserLog includes functions which provide a single synchronous call.

It is not possible to write to any Automation Runtime system logbook with either UserLog or ArEventLog. Automation Runtime creates one logbook named "User", referenced with "$arlogusr", that permits user writes. This is done automatically with UserLogQuick and UserLogFormat.

Users can create logbooks in an _INIT routine using UserLogCreate and subsequently writing to those logbooks with UserLogMessage.
