UserLog

UserLog is an Automation Studio library of functions for writing to user logbooks synchronously.

UserLog's logging functions provide a essential debugging tool during development and production projects.

This file serves as the documentation of the UserLog library.

NOTE: This is not an official library. UserLog is provided as-is under the GNU GPL v3.0 license agreement.
