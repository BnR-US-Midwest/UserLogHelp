UserLog

UserLog is an Automation Studio library of functions for writing to user logbooks synchronously.

UserLog's functions provide a essential debugging tool for development project and production systems.

This file serves as the documentation of the UserLog library.

NOTE: This is not an official library. UserLog is provided as-is under the GNU GPL v3.0 license agreement. Support for this library is provided through the GitHub Issues List.
