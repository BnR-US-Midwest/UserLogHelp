Additional Notes

Text System

Messages written with the UserLog library are compatible with the Automation Studio text system. User Event Text files can be created with namespace User/EventLog. The Text ID should match the Event ID created of the message written with UserLog. See ArEventLogMakeEventID for more information.

Example

UserLogBasic not only writes a message from code, but can link an additional description through the text system. The following example is a line a Structured Text code.

    UserLogBasic(USERLOG_SEVERITY_INFORMATION, 1234, 'Hello World!');

The Event ID for this example is 1611269330. The severity is 1 (INFORMATION), the facility is 10 (USERLOG_FACILITY), and the code is 1234. See ArEventLogMakeEventID for more information on how to derive the 32-bit Event ID.

Bit(s) Description
0..15 Code 0..65535
16..27 Facility 0..4095
28 Reserved (0 for customized event IDs)
29 Customer (1 for customized event IDs)
30..31 Severity 0..3 (Success, Information, Warning, Error)

A User Event Texts file can be created to assign text to Text IDs (Event IDs). The setup requirements for User Event Text files can be followed here.

The logger will show the following results. Text from code is placed in the ASCII Data of the logbook record while text from the User Texts File is placed in the Description (Binary Data).

The methods in this example can also applied to UserLogAdvanced and UserLogCustom.
