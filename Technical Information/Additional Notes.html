Additional Notes

Text System

Messages written with the UserLog library are compatible with the Automation Studio text system. User Event Text files can be created with namespace User/EventLog. The Text ID must match the Event ID created with UserLog functions. See ArEventLogMakeEventID for more information.

Example

UserLogBasic not only writes a message from code, but can link an additional description through the text system./p>

IEC Structured Text

    UserLogBasic(USERLOG_SEVERITY_INFORMATION, 1234, 'Hello World!');
  • Severity: USERLOG_SEVERITY_INFORMATION (1)
  • Facility: USERLOG_FACILITY (10)
  • Code: 1234
  • Event ID: 1611269330
Bit(s) Description
0..15 Code 0..65535
16..27 Facility 0..4095
28 Reserved (0 for customized event IDs)
29 Customer (1 for customized event IDs)
30..31 Severity 0..3 (Success, Information, Warning, Error)

A User Event Text file can be created to link descriptions to Text IDs (Event IDs). The setup requirements for User Event Text files can be followed here.

The logger will show the following results. Text from code is placed in the ASCII Data of the logbook record while text from the User Texts File is placed in the Description (Binary Data).

The methods in this example can also applied to UserLogAdvanced and UserLogCustom.
